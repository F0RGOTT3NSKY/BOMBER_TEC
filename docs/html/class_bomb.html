Detailed Description

La clase Bomb se encarga de instanciar las explosiones al instante en el que el jugador pone una bomba en el suelo.

Para crear las explosiones se invoca el la funcion de Explode() se instancian las explosiones mediante coroutines. Ademas, si la explosion toca una bomba sin explotar se procede a explotarla instantaneamente.

◆ CreateExplosions()

IEnumerator Bomb.CreateExplosions ( Vector3  direction)

CreateExplosions() es un algoritmo dedicado la creacion de las explosiones.

Para crear estas explosiones se crea un rayo de colision para detectar si existe un bloque en donde no se puede crear una explosion. Sino existe ningun bloque en la LayerMask entonces se instancia una la explosion en la hacia la direccion elegida.

Parameters
directionIndica la direccion a la que se va a crear la explosion.
