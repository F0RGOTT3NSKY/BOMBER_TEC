## Installation
***
Actualmente el juego debe ejecutado dentro de la intefaz de Unity
```
$ git clone https://github.com/F0RGOTT3NSKY/Proyecto_II_BomberTEC.git
$ cd ../path/to/the/file
$ npm install
$ npm start
```
